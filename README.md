# Fake_job_Detection_baseline_pipeline
Detect fake job postings using a clean, end‑to‑end NLP baseline on the Kaggle Fake Job Postings dataset. This repo loads and explores the data, cleans job descriptions (lowercasing, HTML/URL/punctuation/digit removal, stopwords, lemmatization), extracts TF‑IDF features (max_features=5000, unigrams+bigrams), and trains a Logistic Regression.n class
